# 📊 Facebook A/B Testing: Optimizing Ad Bidding

This project presents an end-to-end A/B testing analysis to evaluate Facebook's new **Average Bidding strategy** against the traditional **Maximum Bidding strategy** using Python.

Starting from raw advertising data, I performed data exploration, hypothesis formulation, sample size analysis, feature engineering, and statistical hypothesis testing to determine whether the new bidding strategy improves business performance.

---

## 🔍 Project Highlights

- Data cleaning and exploratory data analysis (EDA)
- Hypothesis testing framework (H₀ & H₁)
- Sample size calculation and discussion
- Feature engineering:
  - Click-Through Rate (CTR)
  - Conversion Rate
  - Purchase Rate
  - Revenue per Impression
  - Revenue per Click
  - Average Order Value (AOV)
- Independent Samples t-tests for continuous metrics
- Two-Proportion z-tests for rate-based metrics
- Business interpretation of statistical results

---

## 📈 Key Findings

- CTR decreased under the new bidding strategy.
- Conversion Rate significantly increased.
- Revenue per Impression, Revenue per Click, and Average Order Value all showed statistically significant improvements.
- Overall earnings increased, indicating better monetization despite fewer clicks.

---

## 🏁 Conclusion

Although the Test group generated fewer clicks, it consistently produced higher-value interactions and greater revenue.

Based on the statistical evidence, the **Average Bidding strategy is recommended** as it delivers stronger overall business performance.
